p11a1 由中位数反求未知数并算四分位数

计算解答 中位数四分位数未知数求解 ★★☆☆

（四分位数·未知数）一组数据 3,5,6,x,8,10 的中位数为 7。

（1）求 x 的值；（2）求这组数据的四分位数 Q1​ 和 Q3​。

📌 知识点总结（点击展开／收起）

📌 知识点总结

知识点说明
中位数（n 偶数）中间两个数的平均值；含未知数时列方程求解
四分位数（n=6）均分两半各 3 个，Q1​= 下半中位数，Q3​= 上半中位数
解方程思想利用"中位数为某值"建立等式，反推未知数

✍️ 解题过程

（1）求 x 的值

n=6 时中位数 = 第 3、4 个数的平均值 =7，即这两数之和 =14。

按 x 的大小分类讨论：

  • 若 x≤6：排序后前四位为 3,5,x,6（x 可能更靠前），无论如何第 3、4 个均为 {3,5,6} 中的两个，最大和 =5+6=11，中位数 ≤5.5=7 ✗
  • 若 6<x<8：排序为 3,5,6,x,8,10，第 3 个 =6，第 4 个 =x。令 26+x​=7⇒x=8，与 x<8 矛盾 ✗
  • 若 x≥8：排序为 3,5,6,8,x,10（或 3,5,6,8,10,x），第 3 个 =6，第 4 个 =8，中位数 =26+8​=7 ✓

∴ x≥8。取 x=8，完整数据从小到大排序为：

3, 5, 6, 8, 8, 10
✅ 答案（1）：x=8（或 x≥8）

（2）求 Q1​ 与 Q3​

n=6，均分两半各 3 个，Q1​ 为下半中位数，Q3​ 为上半中位数：

  • 下半 3 个：3, 5, 6，中位数 =5 ⇒ Q1​=5
  • 上半 3 个：8, 8, 10，中位数 =8 ⇒ Q3​=8
✅ 答案（2）：Q1​=5，Q3​=8
