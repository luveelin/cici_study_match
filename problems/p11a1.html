p11a1 由中位数反求未知数并算四分位数

计算解答 中位数四分位数未知数求解 ★★☆☆

（四分位数·未知数）一组数据 3,5,6,x,8,10 的中位数为 7。

（1）求 x 的值；（2）求这组数据的四分位数 Q1​ 和 Q3​。

📌 知识点总结（点击展开／收起）

📌 知识点总结

知识点说明
中位数（n 偶数）中间两个数的平均值；含未知数时列方程求解
四分位数（n=6）均分两半各 3 个，Q1​= 下半中位数，Q3​= 上半中位数
解方程思想利用"中位数为某值"建立等式，反推未知数

✍️ 解题过程

第一步：利用中位数求 x

共 n=6 个数据，中位数是第 3 个与第 4 个数的平均值。

先按大小排列已知部分。由于 x 未知，需讨论其位置：

  • 若 x≤6：排序为 3,5,[x或6],6,8,10，中位数 = 2第3个+第4个​
  • 若 6<x<8：排序为 3,5,6,x,8,10，中位数 = 26+x​
  • 若 x≥8：排序为 3,5,6,8,[x或8],10，中位数 = 28+第4个​

已知中位数 =7。只有第二种情况（6<x<8）能使中位数合理取到 7：

\dfrac{6 + x}{2} = 7 \Rightarrow 6 + x = 14 \Rightarrow x = 8</div><p>但x = 8时恰好处于边界（归入第三种情况也无妨）：排序为3, 5, 6, 8, 8, 10，中位数=\dfrac{6+8}{2} = 7✓</p><divclass="answer−box"><spanclass="answer−label">✅答案（1）：</span><spanclass="answer−value">x = 8</span></div><h4>第二步：求Q_1与Q_3</h4><p>完整数据排序：</p><divclass="formula−block">3, 5, 6, 8, 8, 10 \quad (n = 6)</div><p>下半3个：3, 5, 6，Q_1 =下半中位数= 5</p><p>上半3个：8, 8, 10，Q_3 =上半中位数= 8</p><divclass="answer−box"><spanclass="answer−label">✅答案（2）：</span><spanclass="answer−value">Q_1 = 5, Q_3 = 8$
