下列四个选项中，y 不是 x 的函数的是（    ）

A. y=2x−7    B. y=x3​

C. y=x2      D. y=±x​

📌 知识点总结（点击展开／收起）

📌 知识点总结

知识点说明
函数定义每个 x 必须对应唯一的 y
± 号的含义y=±x​ 是两个函数，y2=x 不是函数
判别方法垂直于 x 轴的直线与图象最多交一个点

✍️ 解题过程

第一步：理解函数定义

函数定义：对于 x 的每一个确定值，y 都有唯一确定的值与之对应。

判断一个关系是否是函数，关键看：每个 x 是否对应唯一的 y。

第二步：逐项分析

选项表达式对应关系是否函数
Ay=2x−7每个 x 对应唯一 y✓ 是
By=x3​每个 x=0 对应唯一 y✓ 是
Cy=x2每个 x 对应唯一 y✓ 是
Dy=±x​一个 x 对应两个 y（如 x=4→y=±2）✗ 不是
🔑 关键点：y=±x​ 实际上是两个函数 y=x​ 和 y=−x​ 的合写，每个 x 对应两个 y 值，违反函数定义。
✅ 答案：D

题1：下列各式中，y 是 x 的函数的是（    ）

A. y2=x    B. y=∣x∣

C. x2+y2=1    D. ∣y∣=2x

💡 答：B。A、C、D 中一个 x（如 x=1）都对应两个 y（y=±1），不满足"唯一对应"；只有 B 每个 x 对应唯一 y。📄 查看详细解答 →

题2：下列图象（描述）中，y 不是 x 的函数的是（    ）

A. 直线 y=x    B. 圆 x2+y2=4

C. 抛物线 y=x2    D. 双曲线 y=x1​

💡 答：B。圆的图象上，一个 x（如 x=0）对应两个 y（y=±2），垂线检验不通过。📄 查看详细解答 →

题3：某关系给出对应：x 取 1,2,3,4 时，y 依次为 2,4,2,5。判断 y 是否是 x 的函数，并说明理由。

💡 答：是函数。"唯一"指每个 x 对应一个 y；不同 x 可以对应相同 y（2 重复出现没问题）。📄 查看详细解答 →
