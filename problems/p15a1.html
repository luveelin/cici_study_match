p15a1 判断 y 是否为 x 的函数（关系式）

（函数判断进阶）下列各式中，y 是 x 的函数的是（    ）

A. y2=x    B. y=∣x∣

C. x2+y2=1    D. ∣y∣=2x

📌 知识点总结（点击展开／收起）

📌 知识点总结

知识点说明
函数定义对 x 的每一个确定值，y 都有唯一确定的值与之对应
判别要点看一个 x 是否可能对应两个或更多 y；若是，则不是函数
常见反例y2=x、x2+y2=1（圆）、∣y∣=2x 都是"一对多"，不是函数

✍️ 解题过程

第一步：回顾函数定义

函数定义：对于 x 的每一个确定值，y 都有唯一确定的值与之对应。

判断时，只要能找到一个 x 对应两个 y，该关系就不是函数。

第二步：逐项判断

选项表达式一个 x 对应几个 y是否函数
Ay2=xx=1⇒y=±1（两个）✗ 不是
By=∣x∣每个 x 对应唯一 ∣x∣✓ 是
Cx2+y2=1x=0⇒y=±1（两个，是圆）✗ 不是
D∣y∣=2xx=1⇒y=±2（两个）✗ 不是
🔑 关键：A、C、D 都可化为"一个 x 对应 y=±(⋯)"的形式，即一对多；只有 B 满足唯一对应。
✅ 答案：B
