（函数判断进阶）下列图象（描述）中，y 不是 x 的函数的是（    ）

A. 直线 y=x    B. 圆 x2+y2=4

C. 抛物线 y=x2    D. 双曲线 y=x1​

📌 知识点总结（点击展开／收起）

📌 知识点总结

知识点说明
垂线检验作垂直于 x 轴的直线，若与图象最多交于一个点，则 y 是 x 的函数
交于两点说明同一个 x 对应两个 y，不是函数
典型非函数图象圆、椭圆、双曲线的一支除外（整条双曲线是函数）、竖直直线 x=a

✍️ 解题过程

第一步：理解垂线检验

垂线检验：垂直于 x 轴的直线与图象最多交一个点 ⇒ 是函数；若某条垂线交到两个点 ⇒ 不是函数。

第二步：逐项检验

选项图象垂线检验结果是否函数
A直线 y=x任意 x=a 交 1 点✓ 是
B圆 x2+y2=4x=0 时交 (0,2) 与 (0,−2) 两点✗ 不是
C抛物线 y=x2任意 x=a 交 1 点✓ 是
D双曲线 y=x1​x=a(a=0) 交 1 点✓ 是
🔑 关键：圆 x2+y2=4 上，一个 x（如 x=0）对应 y=±2 两个值，垂线检验不通过，不是函数。
✅ 答案：B
