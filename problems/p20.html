📌 知识点总结

知识点说明
表格规律观察 a,b,c 三行的对应关系，寻找统一公式
勾股数满足 a2+b2=c2 的正整数组称为勾股数
完全平方(2a​)2 是连接 a 与 b,c 的关键
代入求值得到 c 关于 a 的表达式后，把 a=90 代入

✍️ 解题过程

第一步：发现 (a,b,c) 满足勾股定理

逐行验证：

62+82=36+64=100=102
82+152=64+225=289=172
102+242=100+576=676=262

每一行都满足 a2+b2=c2，所以 (a,b,c) 是一组勾股数。

第二步：发现 b 与 c 的常数关系

观察 b 与 c 的对应值：

c−b=10−8=2,17−15=2,26−24=2,37−35=2,50−48=2

c−b 恒等于 2，即：

b=c−2

第三步：代入消 b，求 c 关于 a 的表达式

将 b=c−2 代入 a2+b2=c2：

a2+(c−2)2=c2
a2+(c2−4c+4)=c2
a2−4c+4=0
4c=a2+4
c=4a2​+1

第四步：代入 a=90 求 c

c=4902​+1=48100​+1=2025+1=2026
✅ 答案：D. 2026
